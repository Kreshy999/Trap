# STREETWEAR MVP FULL BUILD

- Next.js 15
- TypeScript
- Tailwind CSS
- Prisma ORM

- NextAuth Authentication
- Admin Dashboard

## DEPLOYMENT

### Install
npm install

### Database
npx prisma migrate dev

### Run
npm run dev

### Build
npm run build
